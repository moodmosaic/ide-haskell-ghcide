Haskell language support for Atom, powered by [ghcide](https://github.com/digital-asset/ghcide).

```bash
$ apm install language-haskell atom-ide-ui ide-haskell-ghcide
```

The idea behind this package is that *if or when* ghcide already exports a feature, and that feature is supported by the Language Server Protocol support for Atom, it should work without having to release a new version of this Atom package.
